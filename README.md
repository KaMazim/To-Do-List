# To-Do-List
To-do list app made with <a href="https://www.meteor.com/" target="_blank">Meteor</a>, <a href="https://reactjs.org/" target="_blank">React</a> and <a href="https://www.mongodb.com/" target="_blank">MongoDB</a>.

## Running the app

### Install dependencies

```bash
meteor npm install
```

### Running

```bash
meteor
```
